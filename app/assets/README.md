# Assets Directory

Place static assets in this directory:

- `variant_logo.png` - Company logo (optional)
- Custom CSS files
- Favicon

Dash will automatically serve files from this directory.

## Adding a Logo

1. Add your logo file as `variant_logo.png`
2. Update `theme.py` to reference the logo

## Custom CSS

You can add a `custom.css` file here for additional styling.
Dash will automatically include CSS files from the assets folder.
